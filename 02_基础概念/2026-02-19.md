## 一、K线基础

### 1.1 什么是K线

K线（Candlestick）又称蜡烛图、日本线，起源于日本米市。K线记录每天的开盘价、收盘价、最高价、最低价，直观反映价格变动。

### 1.2 K线构成

```
       最高价
         │
    ┌────┴────┐
    │         │
    │  实体   │  ← 开盘价到收盘价的范围
    │         │
    └────┬────┘
         │
       最低价
```

| 要素 | 说明 |
|------|------|
| 实体 | 开盘价与收盘价之间的矩形 |
| 上影线 | 实体到最高价的线段 |
| 下影线 | 实体到最低价的线段 |

### 1.3 K线颜色

| 颜色 | 含义 |
|------|------|
| 红色（阳线） | 收盘价 > 开盘价（上涨） |
| 绿色（阴线） | 收盘价 < 开盘价（下跌） |

## 三、均线系统

### 3.1 什么是均线

移动平均线（MA）是将一定时期内的价格平均化，形成一条平滑曲线。

**计算公式**：
```
MA(N) = (C1 + C2 + ... + CN) / N
```
其中C为收盘价，N为天数。

### 3.2 常用均线

| 均线 | 周期 | 常用场景 |
|------|------|----------|
| MA5 | 5日 | 短期趋势 |
| MA10 | 10日 | 短线操作 |
| MA20 | 20日 | 中期趋势 |
| MA60 | 60日 | 中长期趋势 |
| MA120 | 120日 | 牛熊分界 |
| MA250 | 250日 | 年线 |

### 3.3 均线用法

| 形态 | 信号 |
|------|------|
| 价格上穿均线 | 买入信号 |
| 价格下穿均线 | 卖出信号 |
| 均线多头排列 | 上涨趋势 |
| 空头排列 | 下跌趋势 |
| 均线金叉 | 短期上穿长期，买入 |

## 四、成交量分析

### 4.1 成交量定义

成交量（Volume）指当日成交的股票数量，单位为"手"（1手=100股）。

### 5.3 KDJ

**随机指标**

| 信号 | 说明 |
|------|------|
| K值 < 20，超卖 | 买入机会 |
| K值 > 80，超买 | 卖出风险 |
| 金叉 | K上穿D，买入 |
| 死叉 | K下穿D，卖出 |

### 5.4布林带（Bollinger Bands）

**中轨 = MA(20)**
**上轨 = 中轨 + 2倍标准差**
**下轨 = 中轨 - 2倍标准差**

用法：
- 价格触及上轨：超买
- 价格触及下轨：超卖
- 布林带收窄：波动率低，可能突破
