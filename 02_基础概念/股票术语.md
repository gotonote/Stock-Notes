# 股票术语大全

> 入门必看！A股市场最常用的股票术语汇总

## 基础术语

### 股票相关

| 术语 | 解释 |
|------|------|
| **A股** | 在中国境内上市的股票，以人民币计价 |
| **B股** | 在中国上市的外币计价股票（已较少） |
| **港股** | 在香港上市的股票 |
| **美股** | 在美国上市的股票 |
| **IPO** | 首次公开募股，企业第一次上市 |
| **上市** | 公司股票可以在交易所买卖 |
| **退市** | 公司股票不再在交易所交易 |

### 交易相关

| 术语 | 解释 |
|------|------|
| **开盘价** | 每天交易开始时的第一笔成交价格 |
| **收盘价** | 每天交易结束时的最后一笔成交价格 |
| **最高价** | 当日股票成交的最高价格 |
| **最低价** | 当日股票成交的最低价格 |
| **涨停价** | 当日股价上涨的最大幅度（A股通常为+10%） |
| **跌停价** | 当日股价下跌的最大幅度（A股通常为-10%） |
| **停牌** | 股票暂时不能交易 |
| **复牌** | 股票恢复交易 |

## 行情术语

### 涨跌停（A股特殊制度）

```
    涨跌幅限制：
    
    前收盘 10元
    
    涨停价 = 10 × 1.10 = 11元
    跌停价 = 10 × 0.90 = 9元
    
    * ST股票涨跌幅为5%
    * 科创板/创业板涨跌幅为20%
```

### 盘面术语

| 术语 | 解释 |
|------|------|
| **红盘** | 收盘价高于开盘价（上涨） |
| **绿盘** | 收盘价低于开盘价（下跌） |
| **平盘** | 收盘价等于开盘价 |
| **高开** | 开盘价高于昨日收盘价 |
| **低开** | 开盘价低于昨日收盘价 |
| **高走** | 股价持续上涨 |
| **低走** | 股价持续下跌 |
| **冲高回落** | 股价快速上涨后回落 |
| **探底回升** | 股价快速下跌后回升 |

### K线术语

| 术语 | 解释 |
|------|------|
| **阳线** | 收盘价 > 开盘价 |
| **阴线** | 收盘价 < 开盘价 |
| **十字星** | 开盘价 = 收盘价 |
| **上影线** | K线实体上方的影线 |
| **下影线** | K线实体下方的影线 |
| **光头光脚** | 没有上下影线的K线 |
| **跳空缺口** | 股价与昨日收盘价之间没有重叠区域 |

## 交易术语

### 订单类型

| 术语 | 解释 |
|------|------|
| **市价单** | 以当前市场价格立即成交 |
| **限价单** | 指定价格成交 |
| **止损单** | 跌破某价位自动卖出 |
| **止盈单** | 涨到某价位自动卖出 |

### 持仓术语

| 术语 | 解释 |
|------|------|
| **持仓** | 投资者持有股票 |
| **仓位** | 投资金额占可用资金的比例 |
| **满仓** | 全部资金买入股票 |
| **空仓** | 没有持有任何股票 |
| **建仓** | 第一次买入股票 |
| **加仓** | 继续买入同一只股票 |
| **减仓** | 卖出一部分股票 |
| **清仓** | 全部卖出股票 |
| **补仓** | 下跌后低价买入降低成本 |

### 账户相关

| 术语 | 解释 |
|------|------|
| **浮盈** | 持仓股票当前盈利（未卖出） |
| **浮亏** | 持仓股票当前亏损（未卖出） |
| **实盈/实亏** | 已经卖出的实际盈利/亏损 |
| **总资产** | 持仓市值 + 现金 |
| **可用资金** | 可以用来买股票的资金 |
| **持仓成本** | 买入股票的平均价格 |

## 市场术语

### 市场状态

| 术语 | 解释 |
|------|------|
| **牛市** | 股票市场整体上涨 |
| **熊市** | 股票市场整体下跌 |
| **震荡市** | 股价在一定区间波动 |
| **结构性行情** | 只有部分股票上涨 |
| **普涨** | 大多数股票上涨 |
| **普跌** | 大多数股票下跌 |

### 资金相关

| 术语 | 解释 |
|------|------|
| **主力** | 资金量大的投资者或机构 |
| **散户** | 资金量小的个人投资者 |
| **北向资金** | 从香港流入A股的资金 |
| **南向资金** | 从A股流入港股的资金 |
| **融资** | 向券商借钱买股票 |
| **融券** | 向券商借股票卖出 |
| **杠杆** | 借钱投资，放大收益和风险 |

## 技术分析术语

### 形态术语

| 术语 | 解释 |
|------|------|
| **突破** | 股价超过前期高点/低点 |
| **跌破** | 股价跌破前期支撑位 |
| **支撑位** | 股价下跌时可能获得支撑的价格 |
| **阻力位** | 股价上涨时可能遇到阻力的价格 |
| **箱体震荡** | 股价在固定区间内波动 |
| **平台整理** | 股价横盘整理 |
| **V型反转** | 快速下跌后快速上涨 |
| **M头** | 两个高点形成的顶部形态 |
| **W底** | 两个低点形成的底部形态 |

### 指标术语

| 术语 | 解释 |
|------|------|
| **均线** | 移动平均线 |
| **金叉** | 短期均线从下往上穿过长期均线 |
| **死叉** | 短期均线从上往下穿过长期均线 |
| **多头排列** | 均线向上发散 |
| **空头排列** | 均线向下发散 |
| **背离** | 股价与指标走势不一致 |

## 基本面术语

| 术语 | 解释 |
|------|------|
| **市盈率(PE)** | 股价/每股收益，衡量估值 |
| **市净率(PB)** | 股价/每股净资产，衡量估值 |
| **ROE** | 净资产收益率，衡量盈利能力 |
| **净利润** | 扣除所有成本后的利润 |
| **营收** | 公司的销售收入 |
| **财报** | 财务报告（年报、季报） |
| **业绩预告** | 公司提前公告业绩 |
| **分红** | 公司给股东发钱 |
| **送转** | 送股或转增股本 |

## 常见操作口语

| 说法 | 含义 |
|------|------|
| **上车** | 买入股票 |
| **下车** | 卖出股票 |
| **割肉** | 亏损卖出 |
| **抄底** | 在低位买入 |
| **追高** | 在高位买入 |
| **站岗** | 买在高点后股价下跌 |
| **解套** | 股票回到买入价 |
| **被套** | 股票亏损 |
| **做T** | 当日买入卖出做差价 |
| **梭哈** | 全部资金买入 |

## 总结

| 类别 | 核心术语 |
|------|----------|
| 价格 | 开盘价、收盘价、最高价、最低价 |
| 涨跌 | 涨停、跌停、红盘、绿盘 |
| 仓位 | 建仓、加仓、减仓、清仓 |
| 趋势 | 突破、支撑、阻力、金叉、死叉 |
| 估值 | PE、PB、ROE |
