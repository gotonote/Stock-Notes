## 二、均线系统
### 2.1 什么是均线
**均线**就是平均价格线，把过去N天的收盘价平均一下。
![5日均线示例](../images/ma_example.png)
💡 5日均线 = 最近5天的"平均体温"
### 2.2 常见均线
```
📊 均线层级：
   
           长期均线            
       120日（半年线）          
        60日（季线）           
   
           中期均线             
         20日（季线）          
         30日                   
   
           短期均线             
          5日（攻击线）        
         10日（操盘线）        
   
```
### 2.3 均线金叉/死叉
```
🔄 均线交叉：
金叉（买入信号）          死叉（卖出信号）
 5日                      5日
                          
      10日                   10日
                             
                             
                    
                          
                          
  上涨趋势                下跌趋势
💡 金叉=短期均线向上穿越长期均线
💡 死叉=短期均线向下穿越长期均线
```
