# K线基础入门

> 股票K线是技术分析的根基，本篇带您认识最常用的价格图表

## 什么是K线

**K线**（又称蜡烛图、阴阳线）是一种反映价格变动的图形，由日本大米商人本间宗久在18世纪发明。

### K线的构成

```
        ┌─────┐
        │  上 │
        │  影 │
   ┌────┤  线 ├────┐
   │    └─────┘    │
   │    ┌─────┐    │
   │    │ 实体 │    │ ← 实体（Body）
   │    └─────┘    │
   └────┬─────┬────┘
        │  下 │
        │  影 │
        └─────┘
```

- **实体**：开盘价与收盘价之间的部分
- **上影线**：实体上方的高点连线
- **下影线**：实体下方的低点连线

### K线分类

| 类型 | 特征 | 含义 |
|------|------|------|
| **阳线** | 收盘价 > 开盘价 | 上涨（通常为红色/绿色） |
| **阴线** | 收盘价 < 开盘价 | 下跌（通常为绿色/红色） |
| **十字星** | 开盘价 = 收盘价 | 多空平衡，观望 |

## 常见K线形态

### 1. 单根K线信号

```
锤子线                 上吊线
    ┌─┐                   ┌─┐
    │ │                   │ │
    │ │                   │ │
────┴─┴────           ───┴─┴───
    ↑                       ↑
  下影长                   上影长
  实体小                   实体小
 (看涨)                   (看跌)
```

- **锤子线**：下影线长，实体在顶部 → 下跌后可能反转上涨
- **上吊线**：上影线长，实体在底部 → 上涨后可能反转向下

### 2. 常见组合形态

| 形态 | 图形 | 信号 |
|------|------|------|
| **早晨之星** | ⭐ + 十字星 + 阳线 | 底部反转信号 |
| **黄昏之星** | 阴线 + 十字星 + ⭐ | 顶部反转信号 |
| **吞没形态** | 大阳线包住小阴线 | 强烈反转信号 |
| **乌云盖顶** | 阴线覆盖阳线 | 下跌信号 |

## K线的时间周期

| 周期 | 用途 | 特点 |
|------|------|------|
| 日K | 短线交易 | 反映每日波动 |
| 周K | 中线分析 | 过滤噪音 |
| 月K | 长线分析 | 看趋势方向 |
| 60分钟 | 盘中交易 | 找入场点位 |
| 5分钟 | 超短线 | 精确买卖点 |

## 如何使用K线

### 基本原则

1. **结合趋势**：在上涨趋势中，锤子线更可信
2. **验证信号**：单根K线需配合成交量确认
3. **看位置**：在高位/低位的信号更有效
4. **多周期结合**：日K看方向，分钟K找买点

### 注意事项

- ❌ 不要单独依靠K线做决策
- ❌ 警惕缩量K线的假信号
- ❌ 震荡市中的K线信号容易失效

## 总结

| 要点 | 记住 |
|------|------|
| 阳线上涨，阴线下跌 | 红涨绿跌（A股） |
| 实体越大，力量越强 | 多空博弈结果 |
| 影线越长，阻力越大 | 试探性买卖 |
| 位置比形态重要 | 高位的锤子不如低位的锤子可靠 |
