## 一、仓位管理基础

### 1.1 什么是仓位

**仓位** = 持仓市值 / 总资金

| 仓位 | 含义 |
|------|------|
| 空仓 | 0% |
| 轻仓 | 1-30% |
| 半仓 | 50% |
| 重仓 | 70-90% |
| 满仓 | 100% |

### 1.2 仓位重要性

- **盈亏同源**：仓位决定收益，也决定风险
- **活着最重要**：留得青山在
- **机会是等的**：有仓位才能抓住机会

## 四、止损止盈

### 4.1 止损

| 方法 | 说明 |
|------|------|
| 固定止损 | 亏损到固定比例（如-8%）卖出 |
| 移动止损 | 盈利后回撤一定比例卖出 |
| 逻辑止损 | 买入逻辑不在时卖出 |

### 4.2 止盈

| 方法 | 说明 |
|------|------|
| 目标止盈 | 盈利到目标位卖出 |
| 移动止盈 | 不断抬高止盈点 |
| 分批止盈 | 分几次卖出 |

### 4.3 止损原则

1. **错了就认**：不要硬扛
2. **小亏大赚**：截断亏损，让利润奔跑
3. **一致性**：每次交易都要设止损
