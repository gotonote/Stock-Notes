## 一、风险类型

### 1.1 投资风险

| 风险 | 说明 |
|------|------|
| 市场风险 | 系统性下跌 |
| 集中风险 | 单一持仓过重 |
| 流动性风险 | 无法及时卖出 |
| 信用风险 | 债券违约 |

### 1.2 非投资风险

| 风险 | 说明 |
|------|------|
| 通胀风险 | 购买力下降 |
| 汇率风险 | 跨境投资 |
| 政策风险 | 监管变化 |

## 二、风险管理工具

### 2.1 VaR

**VaR（风险价值）**：给定置信水平下的最大可能损失。

```
VaR = 资产 × 波动率 × 倍数
```

### 2.2 分散化

- **跨资产分散**：股票+债券+商品
- **跨行业分散**：不同行业配置
- **跨地域分散**：A股+港股+美股

### 2.3 对冲

- **期权保护**：买入看跌期权
- **期货对冲**：股指期货
- **融券做空**：对冲下跌

---

## 三、组合优化

### 3.1 马科维茨理论

```
预期收益 = Σ wi × ri
风险 = √(ΣΣ wi × wj × σij)
```

### 3.2 有效前沿

- 风险最小组合
- 收益最大组合
- 最优组合选择

---

## 四、总结

风险管理要点：

1. **识别风险**：了解可能损失
2. **分散投资**：不要把鸡蛋放一个篮子
3. **设置止损**：控制最大亏损
4. **动态调整**：根据风险调整仓位
