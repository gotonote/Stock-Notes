## 一、基本面分析框架

### 1.1 什么是基本面分析

基本面分析是通过分析公司的财务数据、行业环境、竞争优势等因素，评估公司内在价值的方法。

### 1.2 分析层次

```
宏观（经济周期、政策）→ 行业（景气度、竞争）→ 公司（财务、竞争优势）
```

### 1.3 核心逻辑

- **价格围绕价值波动**：短期偏离，长期回归
- **价值决定价格**：好公司最终会被市场认可
- **安全边际**：买入价低于内在价值

## 五、估值方法

### 5.1相对估值法

**市盈率（PE）**：
```
PE = 股价 / 每股收益
合理PE = 增速（%）× 1~2
```

| 增速 | 合理PE |
|------|--------|
| 10% | 10-20倍 |
| 20% | 20-40倍 |
| 30% | 30-60倍 |

### 5.2 绝对估值法（DCF）

**核心公式**：
```
企业价值 = Σ 未来现金流 / (1+折现率)^n
```

**步骤**：
1. 预测未来自由现金流
2. 确定折现率（WACC）
3. 计算终值
4. 折现求和

### 5.3 估值注意事项

- 不同行业PE差异大
- 高增速不能持续给高PE
- 周期股要反着看

## 六、投资逻辑

### 6.1 好公司标准

| 标准 | 说明 |
|------|------|
| 好行业 | 空间大、壁垒高、景气向上 |
| 好生意 | 商业模式好、赚钱容易 |
| 好公司 | 竞争优势强、管理优秀 |
| 好价格 | 估值合理、有安全边际 |

### 6.2 价值投资要点

1. **以合理价格买入**：不是越便宜越好
2. **长期持有**：陪伴好公司成长
3. **逆向投资**：人多的地方不去
4. **安全边际**：留有容错空间

### 6.3 常见误区

| 误区 | 正确做法 |
|------|----------|
| 买便宜货 | 看估值而非价格 |
| 追热点 | 坚守能力圈 |
| 频繁交易 | 长期持有 |
| 过度集中 | 适度分散 |
