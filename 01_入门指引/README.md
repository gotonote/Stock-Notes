## 一、什么是股票

### 1.1 股票的定义

股票是股份公司发行的所有权凭证，代表持有者对公司的部分所有权。购买股票即成为公司的股东，享有公司盈利分红和重大决策投票权。

### 1.2 股票的特点

| 特点 | 说明 |
|------|------|
| 权益性 | 享有股东权益（分红、投票） |
| 流动性 | 可在二级市场交易 |
| 风险性 | 价格波动，可能亏损 |
| 收益性 | 可能的资本利得和分红收益 |

### 1.3 股票与债券的区别

| 对比 | 股票 | 债券 |
|------|------|------|
| 性质 | 所有权凭证 | 债权凭证 |
| 收益 | 不固定（分红+价差） | 固定（利息） |
| 风险 | 较高 | 较低 |
| 期限 | 无期限 | 有固定期限 |

## 四、交易规则

### 4.1 交易单位

- **最小买入**：100股（1手）
- **价格最小变动**：0.01元
- **涨跌幅限制**：±10%（ST股±5%）

### 4.2 交易费用

| 费用 | 收取方 | 费率 |
|------|--------|------|
| 佣金 | 券商 | 万分之1-3 |
| 印花税 | 税务机关 | 卖出千分之一 |
| 过户费 | 交易所 | 万分之0.1 |
| 规费 | 证监会 | 万分之0.2 |

### 4.3 交易方式

- **市价委托**：立即以最优价格成交
- **限价委托**：指定价格成交
- **止损委托**：价格达到止损点自动卖出
