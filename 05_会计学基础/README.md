## 三、利润表

### 3.1 利润表结构

```
营业收入
- 营业成本
= 毛利润
- 销售费用
- 管理费用
- 财务费用
= 营业利润
+ 营业外收入
- 营业外支出
= 利润总额
- 所得税
= 净利润
```

### 3.2 核心指标

| 指标 | 计算 | 意义 |
|------|------|------|
| 毛利率 | (收入-成本)/收入 | 盈利能力 |
| 净利率 | 净利润/收入 | 最终盈利 |
| 营业利润率 | 营业利润/收入 | 主业盈利 |

### 3.3 盈利能力分析

| 指标 | 优秀 | 良好 | 一般 |
|------|------|------|------|
| 毛利率 | >40% | 20-40% | <20% |
| 净利率 | >15% | 5-15% | <5% |

## 四、现金流量表

### 4.1 现金流量分类

| 类型 | 说明 |
|------|------|
| 经营活动 | 主业收支 |
| 投资活动 | 购置资产、对外投资 |
| 筹资活动 | 融资、分红 |

### 4.2 现金流转规律

```
经营现金  投资现金  筹资现金
                        
  ←←←←←←←←←←←←←←←←←←←←
```

### 4.3 现金流肖像

| 类型 | 特征 | 企业状态 |
|------|------|----------|
| +++ | 净现金持续流入 | 优质企业 |
| ++- | 经营、投资流入，筹资流出 | 成熟企业 |
| +-+ | 经营流入，投资、筹资流出 | 成长期 |
| --- | 全面净流出 | 风险企业 |
