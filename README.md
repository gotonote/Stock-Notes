## 简介

本仓库系统记录股票投资学习内容，由浅入深，适合零基础入门到进阶学习。

> ⚡ **持续更新中** - 欢迎 Star & Fork ⭐

---

## 目录结构（由浅入深）

```
Stock-Notes/
├── 01_入门指引/      # 什么是股票、术语科普、发展历程
├── 02_基础概念/      # 基本术语、K线、均线、成交量
├── 03_经济学基础/    # 供需关系、边际效应、机会成本
├── 04_宏观经济学/    # GDP、通胀、利率、汇率、货币政策、财政政策
├── 05_会计学基础/    # 资产负债表、利润表、现金流量表
├── 06_基本面分析/    # 财报解读、行业分析、公司估值
├── 07_技术分析/      # 形态理论、指标分析、量价关系
├── 08_交易策略/      # 趋势跟踪、均值回归、量化策略
├── 09_仓位管理/      # 仓位控制、风险敞口、止损止盈
├── 10_心理建设/      # 情绪控制、认知偏差、交易纪律
├── 11_数学基础/      # 概率统计、期望收益、凯利公式
├── 12_行为金融学/    # 认知偏差、市场非理性、动物精神
├── 13_行业研究/      # 产业链分析、景气度轮动
├── 14_风险管理/      # VaR、分散化、组合优化
├── 15_全球市场/      # A股、港股、美股、ETF配置
├── 16_基金理财/      # 指数基金、ETF、FOF
├── 17_衍生品/        # 期权、期货、可转债
├── 18_前沿动态/      # 最新市场动态、行业新闻
├── 19_每日分析/      # 每日市场复盘、板块轮动、资金流向
├── 20_资源汇总/      # 工具推荐、书籍、开源数据
└── 21_实战案例/      # 实盘记录、复盘分析
```
