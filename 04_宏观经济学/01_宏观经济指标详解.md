## 一、国内GDP数据

### 数据来源
- **中国**：国家统计局 (http://www.stats.gov.cn)
- **美国**：Bureau of Economic Analysis (BEA)

### 解读要点

```
GDP增速 = (本期GDP - 上期GDP) / 上期GDP × 100%
```

**实战解读**：

| GDP增速 | 经济状态 | 市场反应 |
|---------|----------|----------|
| >8% | 高速增长 | 牛市基调 |
| 6-8% | 中速增长 | 震荡上行 |
| 4-6% | 增速放缓 | 结构性行情 |
| <4% | 增长乏力 | 防守为主 |

### 预期差分析

```python
# 实际GDP vs 预期GDP
实际值 - 预期值 > 0.5% → 利好股市
实际值 - 预期值 < -0.5% → 利空股市
```

## 二、CPI 消费者价格指数

### 什么是CPI

CPI衡量居民生活成本的变化，即"物价涨了多少"。

### 构成权重（以中国为例）

| 类别 | 权重 | 举例 |
|------|------|------|
| 食品烟酒 | 30% | 猪肉、蔬菜 |
| 居住 | 20% | 房租、水电 |
| 交通通信 | 15% | 汽油、机票 |
| 教育文化 | 15% | 学费、培训 |
| 医疗保健 | 10% | 药品、诊疗 |
| 其他 | 10% | 服装、日用品 |

### 解读要点

```
CPI < 2%：通缩风险 → 央行可能降息放水
CPI 2-4%：温和通胀 → 经济健康增长
CPI 4-6%：较高通胀 → 央行可能加息
CPI > 6%：恶性通胀 → 警惕政策收紧
```

### 猪肉CPI的特殊性

中国CPI中猪肉权重很高，因此有"猪周期"说法：
- 猪肉价格上涨 → CPI上涨 → 可能引发通胀担忧
- 观察CPI时需区分"猪通胀"和"全面通胀"

## 三、PPI 生产者价格指数

### 什么是PPI

PPI衡量工业企业产品出厂价格变化，反映上游成本端压力。

### 解读要点

```
PPI上涨 → 企业成本上升 → 利润率承压
PPI下跌 → 成本下降 → 企业利润空间扩大
```

### PPI与CPI的关系

| 关系 | 含义 |
|------|------|
| PPI先于CPI上涨 | 成本推动型通胀 |
| CPI先于PPI上涨 | 需求拉动型通胀 |
| PPI和CPI背离 | 结构性问题 |

### 投资应用

- **PPI上行期**：关注能够转嫁成本的企业（资源、垄断）
- **PPI下行期**：关注中下游制造企业（成本降低）

## 四、PMI 采购经理指数

### 什么是PMI

PMI是通过问卷调查得出的景气度指标，50为荣枯线。

### 解读要点

```
PMI > 50：经济扩张
PMI < 50：经济收缩
PMI = 50：无变化
```

### PMI细分

| 指标 | 说明 |
|------|------|
| 综合PMI | 整体经济状况 |
| 制造业PMI | 工业生产状况 |
| 非制造业PMI | 服务业状况 |

### 投资应用

- **PMI连续3个月>50**：经济复苏确认，增配股票
- **PMI连续3个月<50**：经济走弱，减配股票
- **PMI拐点**：往往预示股市拐点

## 五、社融与M2（金融数据）

### 什么是社融

**社会融资规模**：实体经济从金融体系获得的资金总量。

### 社融数据结构

| 组成部分 | 说明 |
|----------|------|
| 人民币贷款 | 企业和居民贷款 |
| 外币贷款 | 美元等外币贷款 |
| 委托贷款 | 银行代理贷款 |
| 信托贷款 | 信托公司贷款 |
| 企业债券 | 信用债融资 |
| 政府债券 | 地方政府专项债 |
| 股票融资 | 股权融资 |

### 解读要点

```
社融增速上升 → 实体经济获得资金增加 → 经济增长动力增强 → 利好股市
社融增速下降 → 信用收缩 → 经济下行压力 → 利空股市
```

### 什么是M2

**M2（广义货币）**：流通中的现金+企业存款+居民储蓄。

```
M2增速 ≈ 货币供应增速
M2增速过高 → 通胀预期
M2增速过低 → 流动性紧张
```

### 社融与M2的关系

| 社融-M2差值 | 含义 |
|-------------|------|
| 差值扩大 | 实体经济融资活跃 |
| 差值收窄 | 资金在金融体系空转 |

## 六、LPR 贷款市场报价利率

### 什么是LPR

LPR是商业银行贷款利率的参考基准，由18家银行报价形成。

### LPR结构

```
LPR = MLF利率 + 加点
```

### 解读要点

```
LPR下调 → 贷款成本下降 → 利好购房/企业投资 → 利好股市
LPR上调 → 贷款成本上升 → 抑制过热 → 利多债市
```

### LPR历史走势

- **2019年8月**：改革形成新LPR机制
- **2022年**：多次下调，稳增长
- **2024年**：LPR继续下行

## 九、实战：数据解读流程

### 第一步：判断方向

```
数据 > 预期 → 利好该数据相关资产
数据 < 预期 → 利空该数据相关资产
```

### 第二步：评估幅度

```
超预期幅度 > 0.5% → 显著影响
超预期幅度 0.2-0.5% → 轻度影响
超预期幅度 < 0.2% → 影响有限
```

### 第三步：判断趋势

```
单月数据异常 → 可能是噪音
连续3个月同向 → 趋势确认
```

### 第四步：政策响应

```
数据差 → 政策宽松预期 → 股市可能反弹
数据好 → 政策收紧预期 → 股市可能承压
```
