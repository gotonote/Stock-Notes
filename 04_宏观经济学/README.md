## 一、GDP国内生产总值

### 1.1 GDP定义

**GDP（Gross Domestic Product）**：一定时期内，一个国家或地区生产的所有最终商品和服务的市场价值。

### 1.2 GDP构成

```
GDP = 消费(C) + 投资(I) + 政府支出(G) + 净出口(NX)
```

| 组成部分 | 说明 | 对股市影响 |
|----------|------|------------|
| 消费 | 居民消费支出 | 消费股业绩 |
| 投资 | 企业投资 | 周期股业绩 |
| 政府支出 | 财政政策 | 基建股走势 |
| 净出口 | 出口-进口 | 出口导向型公司 |

### 1.3 GDP与股市关系

| GDP走势 | 经济状态 | 股市表现 |
|---------|----------|----------|
| GDP上行 | 经济繁荣 | 牛市 |
| GDP下行 | 经济衰退 | 熊市 |
| GDP企稳 | 经济触底 | 反弹 |

### 1.4 投资应用

- **GDP上行期**：配置周期股、金融股
- **GDP下行期**：配置消费股、避险资产
- **GDP拐点**：关注宏观经济周期转换

## 二、通货膨胀

### 2.1 通胀定义

**通货膨胀**：物价总水平持续上涨，货币购买力下降。

### 2.2 通胀类型

| 类型 | 原因 | 对股市影响 |
|------|------|------------|
| 需求拉动 | 需求过旺 | 利好股市 |
| 成本推动 | 成本上升 | 利空股市 |
| 货币超发 | 流动性过剩 | 利好股市 |

### 2.3 通胀指标

| 指标 | 说明 |
|------|------|
| CPI | 消费者价格指数（居民生活成本） |
| PPI | 生产者价格指数（工业品价格） |

### 2.4 投资应用

| 通胀阶段 | 配置方向 |
|----------|----------|
| 低通胀 | 成长股 |
| 温和通胀 | 周期股 |
| 高通胀 | 资源股、黄金 |
| 恶性通胀 | 现金为王 |
