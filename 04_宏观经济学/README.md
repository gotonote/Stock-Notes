## 一、GDP国内生产总值

### 1.1 GDP定义

**GDP（Gross Domestic Product）**：一定时期内，一个国家或地区生产的所有最终商品和服务的市场价值。

### 1.2 GDP核算

```
GDP = 消费(C) + 投资(I) + 政府支出(G) + 净出口(NX)
```

| 组成部分 | 说明 | 对股市影响 |
|----------|------|------------|
| 消费(C) | 居民消费支出 | 消费股业绩 |
| 投资(I) | 企业投资 | 周期股业绩 |
| 政府支出(G) | 财政政策 | 基建股走势 |
| 净出口(NX) | 出口-进口 | 出口导向型公司 |

### 1.3 GDP与股市关系

| GDP走势 | 经济状态 | 股市表现 |
|---------|----------|----------|
| GDP上行 | 经济繁荣 | 牛市 |
| GDP下行 | 经济衰退 | 熊市 |
| GDP企稳 | 经济触底 | 反弹 |
| GDP拐点 | 周期切换 | 重大机会 |

### 1.4 投资应用

- **GDP上行期**：配置周期股、金融股、地产股
- **GDP下行期**：配置消费股、避险资产、公用事业
- **GDP拐点**：关注宏观经济周期转换，逆向布局
- **GDP数据公布**：每季度公布，关注实际值与预期的差异

## 二、通货膨胀

### 2.1 通胀定义

**通货膨胀**：物价总水平持续上涨，货币购买力下降。

### 2.2 通胀类型

| 类型 | 原因 | 对股市影响 |
|------|------|------------|
| 需求拉动型 | 需求过旺，供不应求 | 利好股市，企业盈利增加 |
| 成本推动型 | 原材料成本上升 | 利空股市，压缩利润 |
| 货币超发型 | 流动性过剩 | 利好股市，资产价格上涨 |
| 输入型 | 进口商品涨价 | 取决于传导机制 |

### 2.3 通胀指标

| 指标 | 说明 | 投资意义 |
|------|------|----------|
| CPI | 消费者价格指数（居民生活成本） | 民生相关 |
| PPI | 生产者价格指数（工业品价格） | 上游原材料 |
| PMI | 采购经理指数（景气度） | 先行指标 |
| PCE | 个人消费支出价格指数（美联储关注） | 美国通胀锚 |

### 2.4 通胀与利率关系

```
通胀上升 → 央行加息 → 流动性收紧 → 股市承压
通胀下降 → 央行降息 → 流动性宽松 → 股市利好
```

### 2.5 投资应用

| 通胀阶段 | 经济状态 | 配置方向 |
|----------|----------|----------|
| 低通胀(<2%) | 经济疲软 | 成长股、债券 |
| 温和通胀(2-4%) | 经济正常 | 周期股、股票 |
| 高通胀(4-6%) | 经济过热 | 资源股、黄金、能源 |
| 恶性通胀(>6%) | 经济失控 | 现金为王、实物资产 |

### 2.6 实际案例

**2021-2022年美国通胀**：
- CPI从4%飙升至9%
- 美联储激进加息
- 科技股大幅回调
- 能源股逆势上涨

## 三、利率

### 3.1 利率类型

| 利率 | 说明 | 影响 |
|------|------|------|
| 存款基准利率 | 银行存贷款利率的锚 | 传统货币政策工具 |
| MLF利率 | 中期借贷便利利率 | 央行调控意图 |
| LPR | 贷款市场报价利率 | 实际贷款利率 |
| SLF利率 | 常备借贷便利利率 | 利率走廊上限 |
| 美联储利率 | 美元利率基准 | 全球资产定价锚 |

### 3.2 利率与股市关系

```
利率 ↓ 股价 ↑ （流动性宽松，融资成本降低）
利率 ↑ 股价 ↓ （流动性收紧，估值承压）
```

### 3.3 利率传导机制

```
1. 央行调整政策利率
        ↓
2. 银行存贷款利率变化
        ↓
3. 企业融资成本变化
        ↓
4. 居民消费意愿变化
        ↓
5. 经济活动和通胀变化
        ↓
6. 资产价格变化
```

### 3.4 利率对不同行业的影响

| 影响 | 行业 |
|------|------|
| 利好（降息） | 地产、银行、券商、有色 |
| 利空（加息） | 高估值成长股、科技股 |
| 影响中性 | 消费、公用事业 |

### 3.5 投资应用

- **降息周期**：利好股市，特别是高负债行业（地产、金融）
- **加息周期**：利空股市，特别是高估值成长股
- **利率拐点**：关注货币政策转向信号，预判市场底部顶部
- **利率预期**：市场预期比实际政策更重要

## 六、汇率

### 6.1 汇率定义

**汇率**：一种货币兑换另一种货币的比率。

### 6.2 汇率影响因素

| 因素 | 影响 |
|------|------|
| 经济基本面 | 经济增长、利率差异 |
| 货币政策 | 利差变化 |
| 贸易收支 | 进出口 |
| 市场预期 | 情绪和资本流动 |
| 地缘政治 | 避险需求 |

### 6.3 汇率对股市影响

| 汇率走势 | 影响 | 受益板块 |
|----------|------|----------|
| 本币升值 | 利好进口、美元负债企业 | 航空、金融地产 |
| 本币贬值 | 利好出口企业 | 出口导向型 |
| 汇率稳定 | 有利于投资 | 整体市场 |

### 6.4 人民币汇率投资意义

```
人民币升值 → 外资流入 → 利好A股
人民币贬值 → 外资流出 → 资金面承压
```

### 6.5 投资应用

- **人民币升值期**：配置金融地产、航空、造纸
- **人民币贬值期**：配置出口导向型企业（电子、纺织）
- **汇率波动期**：关注外汇风险，对冲汇率波动
