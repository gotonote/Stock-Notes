## 一、供需关系

### 1.1 需求定律

**需求定理**：在其他条件不变的情况下，商品的需求量与价格呈反向变动关系。

```
价格↑ → 需求量↓
价格↓ → 需求量↑
```

### 1.2 供给定律

**供给定理**：在其他条件不变的情况下，商品的供给量与价格呈正向变动关系。

```
价格↑ → 供给量↑
价格↓ → 供给量↓
```

### 1.3 供需均衡

- **均衡价格**：需求=供给时的价格
- **供不应求**：价格上涨（通胀压力）
- **供过于求**：价格下跌（通缩风险）

### 1.4 投资应用

| 市场情况 | 对股市影响 |
|----------|------------|
| 供不应求 | 利好相关公司股价 |
| 供过于求 | 利空相关公司股价 |
| 供需平衡 | 市场稳定 |

## 二、边际分析

### 2.1 边际效用

**边际效用**：增加一单位商品消费所带来的效用增量。

```
边际效用 = 总效用增量 / 消费增量
```

**边际效用递减规律**：随着消费数量增加，每增加一单位商品带来的效用逐渐减少。

### 2.2 边际成本

**边际成本**：增加一单位产量所带来的成本增量。

```
边际成本 = 成本增量 / 产量增量
```

### 2.3 投资应用

- **边际思维**：关注增量变化，而非总量
- **边际改善**：公司业绩边际好转比绝对值更重要
- **边际定价**：市场边际定价决定股价

## 三、机会成本

### 3.1 定义

**机会成本**：为了获得某种收益而放弃的次优选择所带来的收益。

```
机会成本 = 放弃的最佳替代选项收益
```

### 3.2 示例

| 决策 | 机会成本 |
|------|----------|
| 持有现金 | 错过股票上涨收益 |
| 持有股票 | 错过债券利息收益 |
| all in 单一股票 | 分散化收益 |

### 3.3 投资应用

- **仓位决策**：满仓的机会成本是抄底资金
- **持币vs持股**：比较预期收益和机会成本
- **风险收益比**：收益必须覆盖机会成本
