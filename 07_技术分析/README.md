## 一、技术分析基础

### 1.1 什么是技术分析

技术分析是通过研究历史价格和成交量数据，预测未来股价走势的方法。

### 1.2 三大假设

1. **价格反映一切**：所有信息都反映在价格中
2. **价格呈趋势运动**：趋势会延续
3. **历史会重演**： patterns会重复出现

### 1.3 分析对象

| 对象 | 说明 |
|------|------|
| 价格 | 开盘、收盘、最高、最低 |
| 成交量 | 成交数量 |
| 持仓量 | 未平仓合约 |
| 资金流向 | 主力资金进出 |
