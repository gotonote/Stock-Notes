## 一、概率统计

### 1.1 概率思维

**核心观点**：投资是概率游戏，追求长期大概率盈利。

| 概念 | 说明 |
|------|------|
| 胜率 | 盈利交易占比 |
| 赔率 | 盈利/亏损比例 |
| 期望值 | 胜率×赔率 - 失败率×1 |

### 1.2 期望值公式

```
E = p × r - (1-p) × 1
```

| 情况 | 结果 |
|------|------|
| E > 0 | 正期望，值得投资 |
| E < 0 | 负期望，不值得投资 |
| E = 0 | 无优势，不做 |

## 二、凯利公式

### 2.1 公式

```
f* = (bp - q) / b
```

| 符号 | 含义 |
|------|------|
| f* | 应押注的资金比例 |
| b | 赔率（盈亏比） |
| p | 胜率 |
| q | 1-p |

### 2.2 应用

| 胜率 | 赔率 | 建议仓位 |
|------|------|----------|
| 60% | 1:1 | 20% |
| 70% | 1:1 | 40% |
| 50% | 2:1 | 25% |

### 2.3 注意

- 凯利公式是理想模型
- 实际要用减半或更保守的仓位
